입력상태를 실시간으로 확인하는 것을 이용하여, 실시간으로 변하는 가변저항값을 세그먼트에 옮겨서 가변저항값이 어떻게 나오는지 확인할 수 있었다. 이때, 가변저항값에 우리가 통제할 수 없는 여러가지 외란에 의해 레버를 돌리지 않더라고 미세한 떨림이 있는 것을 확인할 수 있다.
